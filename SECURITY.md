# Security

Memory Daymaker reads local OpenClaw session transcripts and writes local memory files. It should not execute shell commands or read credentials.

If you find a security issue, please do not include private transcripts, tokens, or chat identifiers in a public report. Once this repository is public, prefer GitHub private vulnerability reporting if it is enabled for the repository.

For non-sensitive bugs, open a normal GitHub issue with:

- OpenClaw version
- Memory Daymaker version
- installation method
- sanitized config snippet
- sanitized logs or test output
